## Quick Start

1. Add a PostgreSQL database in Railway
2. Configure environment variables (see below)
3. Railway will auto-deploy on git push

## Required Environment Variables

```bash
DATABASE_URL=<auto-generated-by-railway-postgres>
JWT_SECRET=<your-secret>
VITE_SUPABASE_URL=<your-supabase-url>
VITE_SUPABASE_ANON_KEY=<your-supabase-key>
RESEND_API_KEY=<your-resend-key>
RESEND_FROM_EMAIL=erayawellnesstravels@gmail.com
STRIPE_SECRET_KEY=<your-stripe-key>
STRIPE_PUBLISHABLE_KEY=<your-stripe-pk>
STRIPE_WEBHOOK_SECRET=<your-stripe-webhook>
ADMIN_EMAIL=erayawellnesstravels@gmail.com
NODE_ENV=production
```

## Optional Variables

```bash
GEMINI_API_KEY=<for-ai-chat>
OPENAI_API_KEY=<alternative-ai>
```

## Deployment

Push to main branch - Railway will automatically build and deploy.
